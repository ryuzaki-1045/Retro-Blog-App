# Retro-Blog-App

**This is a Blog App directly from the 2010's or earlier ig**

## Features

- User authentication
- Create, read, update, and delete posts

## Tech-Stack

- React+Tailwind(just for sizing and continerization)
- Appwrite for BaaS

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Retro-Blog-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Retro-Blog-App
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run:

```bash
npm run dev
```

Open your browser and go to `http://localhost:3000` to see the app in action.
